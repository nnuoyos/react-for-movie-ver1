# movie site with Create React App

### 💻프로젝트 소개
실시간 영화 리스트를 볼 수 있는 영화 사이트

### ⏰작업 기간
- 1 weeks

### 📌주요 기능
1. create-react-app 사용
2. YTS torrents API 이용하여 영화 차트 리스트 설정
3. 평점 8.5 점 이상의 영화 리스트 나열

[기능]
- 디테일 페이지에서 home 로고 클릭 시 home 화면으로 이동
- 로딩 페이지 설정


### ⚙개발 환경
- skill : react.js
- IDE : vscode
- framework : bootstrap slick
